# Python - Sincronização com Semáforos Avançada

## 📖 Objetivo

Aprofundar os conceitos de sincronização em programação concorrente utilizando semáforos em Python, aplicando técnicas mais avançadas para o controle de acesso a recursos compartilhados e coordenação entre processos e threads.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Utilização de múltiplos semáforos;
- Sincronização avançada entre processos e threads;
- Controle de regiões críticas;
- Coordenação de execução concorrente;
- Resolução de problemas de sincronização.

## 💻 Tecnologias

- Python 3

## ▶️ Como executar

1. Instale o Python 3 em seu computador.
2. Clone este repositório ou faça o download dos arquivos.
3. Execute o arquivo desejado pelo terminal:

```bash
python nome_do_arquivo.py
```
